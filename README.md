# Serac Engine

Serac Engine is a game engine written from scratch in modern C++23 and SFML as a long-term exploration of game engine architecture, real-time rendering, gameplay systems, tools, and performance-oriented software design.

The project is developed incrementally, with each subsystem implemented and documented as it matures. The goal is to understand and build the core technologies behind modern game engines rather than simply using existing frameworks.

## Philosophy

Rather than optimizing for rapid feature development, Serac emphasizes:

- Modern C++ design
- Readable and maintainable architecture
- Performance-conscious implementation
- Learning through implementation
- Minimal unnecessary dependencies
